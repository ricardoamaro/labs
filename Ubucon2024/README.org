* Hello World Snap

** Build hello world snap

https://www.gtk.org/docs/getting-started/hello-world

#+begin_src bash
cd hello-world-gtk
pwd
snapcraft pack -v --debug

#+end_src

#+RESULTS:

** Install

#+begin_src bash
sudo snap install hello-world-gtk_0.1_amd64.snap --dangerous
#+end_src

** Run

#+begin_src bash
snap run hello-world-gtk
#+end_src
